# Monthly Billing System (Python)

This project is a Python-based billing system that calculates monthly revenue based on item usage duration.

## Features
- Calculates active days of items within a given month
- Applies prorated billing based on usage
- Groups similar items
- Generates total monthly revenue

## Tech Used
- Python
- datetime module
- collections (defaultdict)

## How it Works
- Input: List of items with start and end dates
- Process: Calculate active days and billing amount
- Output: Line items and total revenue

## Run
python billing.py
